## 🚀 Getting Started

The setup process requires a few simple steps. You need the original game files if you own a retail copy of the game. If you have the legitimate digital files, this tool helps you launch the game on your system.

1. Visit the [official releases page](https://github.com/slippery-familyceratopsidae440/cnc-port/releases) to download the latest version of the software.
2. Select the file marked for your specific operating system. For Windows users, look for the file ending in .exe.
3. Save this file to a folder on your computer.
4. Extract the contents of the download if it comes in a compressed folder.
5. Place your original game data files into the main folder of the port. The game needs these files to render the maps and units.
6. Run the executable file to start the game.

## 🛠 Troubleshooting

If the game does not start, verify that you placed the required data files in the correct folder. The system looks for specific file names from the original installation media. Ensure your folder permissions allow the application to read these files.

If the screen appears black, check your monitor resolution settings. You can edit the configuration file inside the main folder to force a specific window size. Open the file named settings.ini with a text editor to change the resolution values. Save the file and restart the game to apply these updates.

Run the game as an administrator if updates fail to save correctly. Right-click the application icon and select Run as Administrator to grant the necessary access.
